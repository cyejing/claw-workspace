# 2026-03-16 工作日志

## 微信公众号文章抓取问题

**问题：** 微信公众号文章无法通过自动化工具抓取
- URL: https://mp.weixin.qq.com/s/FrWhkYKonLX3ZuD6nDhXpw
- 标题: OpenClaw 进入"日更级"狂飙模式：刚刚、2026.3.12 版本发布
- 来源: 云头条

**原因：** 微信 POC (Proof of Concept) 验证机制会检测：
1. 浏览器指纹
2. 自动化行为模式
3. 网络环境异常

即使使用 Scrapling StealthyFetcher（模拟真实浏览器）也会被检测到，返回 `wappoc_appmsgcaptcha` 验证页面。

**解决方案：**
1. 直接在浏览器中打开链接查看
2. 手动复制文章内容发送给助手
